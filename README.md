## <a name="parte1">01 - Ferramentas e servidor</a>

- node: https://nodejs.org/en/
- https://www.npmjs.com/

-> npm init

```json
{
  "name": "aula01",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "dev": "lite-server"
  },
  "author": "josemalcher",
  "license": "ISC",
  "devDependencies": {
    "lite-server": "^2.3.0"
  }
}

```

-> npm run dev

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Aula 01</title>
</head>
<body>
<script>
    var nome = "josemalcher";
    console.log(nome);
</script>
</body>
</html>
```
